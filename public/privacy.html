Política de Privacidade

Última atualização: 26 de Outubro de 2025

1. Introdução

O SyncAds respeita sua privacidade e está comprometido com a proteção dos seus dados pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD).

2. Dados Coletados

2.1 Dados fornecidos por você

  • Nome completo
  • Email
  • Senha (criptografada)
  • CPF (opcional)
  • Data de nascimento (opcional)
  • Configurações de conta

2.2 Dados coletados automaticamente

  • IP address
  • User Agent
  • Timestamp de requisições
  • Métricas de performance
  • Endpoints acessados

2.3 Dados de uso

  • Mensagens de chat (armazenadas localmente)
  • Conteúdo gerado pela IA
  • Arquivos baixados
  • Preferências de configuração

3. Como Usamos Seus Dados

Finalidades:
  • Prestar o serviço de chat com IA
  • Melhorar a experiência do usuário
  • Prevenir abusos e garantir segurança
  • Cumprir obrigações legais
  • Analisar métricas de uso

4. Retenção de Dados

Mantemos seus dados:

  • Conta ativa: Durante todo o período de uso
  • Conta inativa: Até 1 ano após última atividade
  • Por lei: Conforme exigido por legislação
  • Backup: Até 90 dias após exclusão

5. Compartilhamento de Dados

NÃO vendemos seus dados. Compartilhamos apenas:

  • Com provedores de IA (para processamento de mensagens)
  • Com Supabase (hospedagem de banco de dados)
  • Com autoridades, quando exigido por lei

6. Seus Direitos (LGPD)

6.1 Acesso

Você pode solicitar acesso a todos os dados que temos sobre você.

6.2 Correção

Você pode solicitar correção de dados incorretos ou desatualizados.

6.3 Exclusão

Você pode solicitar exclusão de seus dados pessoais (direito ao esquecimento).

6.4 Portabilidade

Você pode solicitar seus dados em formato estruturado.

6.5 Revogação de Consentimento

Você pode retirar seu consentimento a qualquer momento.

6.6 Oposição

Você pode opor-se ao processamento de seus dados.

7. Segurança dos Dados

Medidas de segurança implementadas:

  • 🔐 Criptografia TLS/SSL para dados em trânsito
  • 🔐 Criptografia AES-256 para dados em repouso
  • 🛡️ Rate limiting para prevenir abuso
  • 🛡️ Circuit breaker para resiliência
  • 📊 Monitoramento 24/7
  • 🔑 Autenticação JWT segura
  • 🔒 Row Level Security (RLS) no banco de dados
  • 🧪 Testes de segurança regulares

8. Cookies e Tecnologias Semelhantes

Utilizamos cookies para:

  • Autenticação (sessão)
  • Preferências do usuário
  • Analytics (anônimo)

Você pode desativar cookies nas configurações do navegador.

9. Transferência Internacional

Seus dados podem ser armazenados em servidores fora do Brasil (EUA, Europa). Garantimos segurança equivalente.

10. Proteção de Menores

Nossos serviços são para maiores de 18 anos. Não coletamos dados de menores intencionalmente.

11. Mudanças nesta Política

Podemos atualizar esta política periodicamente. Notificaremos mudanças significativas por email.
